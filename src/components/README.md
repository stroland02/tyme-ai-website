# Components Directory Structure

## `/ui`
Reusable UI components (buttons, inputs, cards, etc.)
- Button.tsx
- Input.tsx
- Card.tsx
- Badge.tsx
- etc.

## `/layout`
Layout components that wrap pages
- Header.tsx
- Footer.tsx
- Navigation.tsx
- Container.tsx

## `/sections`
Page-specific sections
- Hero.tsx
- ServicesOverview.tsx
- Testimonials.tsx
- CTA.tsx
- etc.
